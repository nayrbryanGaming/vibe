How It Works

Three taps.
Permanently on-chain.

No apps to follow, no posts to curate. Just real humans, meeting in real life — recorded on an immutable ledger.

1
Open & Connect Wallet

Launch VIBE and connect your Solana wallet. No account creation, no KYC. Your keypair is your identity — sovereign and portable.

2
NFC Tap

Hold your phone near someone else's. VIBE exchanges wallet addresses over NFC and initiates a mutual Proof-of-Connection transaction.

3
Connection Minted

A cNFT is minted on Solana with your timestamp, GPS location, and both wallet addresses. Immutable. Verifiable. Yours forever.

Live API

Open social graph,
accessible to everyone

The VIBE backend exposes a public REST API. Query any wallet's connection graph, check degrees of separation, or pull heatmap coordinates.

GET /api/health
→ { "status": "ok", "protocol": "SOLTAG v1.0", "graph": { "totalUsers": 50, "totalConnections": 150 } }

GET /api/connections/:wallet
→ { "wallet": "9xDef…", "connections": ["Fg3W…", "HnZa…"], "count": 3 }

GET /api/separation/:walletA/:walletB
→ { "degrees": 2 }

GET /api/heatmap
→ { "dots": [{ "latitude": -6.175, "longitude": 106.827, … }] }

POST /api/connections { walletA, walletB, latitude, longitude, signature }
→ { "success": true }
